# Image-Shrink
Desktop app for minimising images

## Usage

### Install Dependencies

```
npm install
```

### Run

```
npm start (without Nodemon)
npm run dev  (with Nodemon)
```

### Package

```
npm run package-mac     (For Mac)
npm run package-win     (For Windows)
```
